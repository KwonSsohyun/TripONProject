## 👀 개요  
일상에서 벗어나 OFF, 여행만을 즐길 수 있길 바라는 ON  
‘tripON’ 브랜드의 마음을 담은 여행 웹 어플리케이션 서비스  

<br>

## 📋 제작기간 & 참여인원  
· 2022.04.04 ~ 2022.05.11  
· 8명
